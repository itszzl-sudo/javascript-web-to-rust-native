# 架构设计

## 1. 项目概述

JavaScript-Web-to-Rust-Native 将任意打包后的 JavaScript 代码编译为原生 Rust 二进制程序。

### 1.1 核心原则

> **"浏览器不关心你的源码是什么框架，只关心最终能否执行 JavaScript"**

| 浏览器关心的 | 浏览器不关心的 |
|-------------|----------------|
| 最终的 JavaScript 代码 | 源码框架 (Vue/React/Svelte) |
| DOM/BOM API 调用 | 打包工具 (Vite/Webpack/Rollup) |
| 事件系统 | 模块系统 (ESM/CJS/UMD) |
| 运行时语义 | JSX/TS/模板语法 |

### 1.2 技术栈

| 组件 | 技术 | 版本 |
|------|------|------|
| JavaScript 解析 | SWC | 最新 |
| 转译器 | jrust-translator | 0.1 |
| 运行时 | jrust-runtime | 0.1 |
| 原生编译 | Cranelift | 最新 |
| 渲染引擎 | rust-browser / Servo | 最新 |
| 语言 | Rust | 1.94+ |

## 3. 核心组件

### 3.1 jrust-translator

JavaScript → Rust 转译器

```
jrust-translator/
├── src/
│   ├── lib.rs           # 库入口
│   ├── ast/             # AST 定义
│   ├── parser.rs        # 解析器
│   ├── swc_parser.rs    # SWC 集成
│   ├── analyzer.rs      # 语义分析
│   ├── codegen.rs       # Rust 代码生成
│   ├── compiler.rs      # 编译主逻辑
│   ├── ir_gen.rs        # Cranelift IR 生成
│   └── cli/             # CLI 工具
└── examples/            # 示例
```

### 3.2 jrust-runtime

JavaScript 运行时环境

```
jrust-runtime/
├── src/
│   ├── core/            # 核心类型 (JsValue, JsObject, JsArray, JsFunction)
│   ├── dom/             # DOM API (Document, Element, Node, Events)
│   ├── bom/             # BOM API (Window, Location, Navigator, Fetch)
│   ├── bindings/        # 函数绑定注册表
│   ├── comm/            # 通信层 (ThreadChannel, ProcessChannel)
│   ├── director/        # 编排器 (JsRustTree)
│   └── compiler/        # 编译器集成
├── tests/               # 单元测试
├── examples/            # 示例程序
└── benches/             # 基准测试
```

### 3.3 cranelift-compiler

Cranelift JIT/AOT 编译器封装

```
cranelift-compiler/
├── src/
│   ├── lib.rs           # 库入口
│   ├── ir.rs            # IR 定义
│   ├── compiler.rs      # 编译逻辑
│   └── linker.rs        # 链接器
```

### 3.4 其他组件

| 组件 | 功能 |
|------|------|
| binding-generator | 过程宏，自动生成 JS → Rust 函数绑定 |
| jrust-servo | Servo 浏览器引擎集成 |
| jrust-browser | rust-browser 项目集成 |
| director | CLI 编排工具，协调编译流程 |
| vue-demo | Vue 应用演示 |

## 4. 通信架构

### 4.1 通信模式

```
┌─────────────────────────────────────────────────────────────┐
│  CommMode 枚举                                              │
├─────────────┬─────────────────┬─────────────────────────────┤
│   Direct    │     Thread      │          Process            │
│  直接调用    │   跨线程通信    │         跨进程通信          │
├─────────────┼─────────────────┼─────────────────────────────┤
│  直接函数   │  mpsc::channel  │    TCP + JSON 序列化        │
│    调用     │   ThreadChannel │    ProcessChannel           │
└─────────────┴─────────────────┴─────────────────────────────┘
```

### 4.2 使用场景

| 场景 | 推荐模式 | 说明 |
|------|---------|------|
| 单线程应用 | Direct | 最简单，直接函数调用 |
| 多线程 UI | Thread | UI 线程与业务逻辑分离 |
| 多进程隔离 | Process | 最高隔离级别，支持崩溃恢复 |

## 7. 工作流程详解

### 7.1 编译时

```
JavaScript 源码
      │
      ▼ (SWC 解析)
ESTree AST
      │
      ▼ (语义分析)
类型信息 + 调用图
      │
      ▼ (代码生成)
Rust 代码 / Cranelift IR
      │
      ▼ (Cargo 编译)
原生二进制
```

### 7.2 运行时

```
用户交互
    │
    ▼
事件系统 → jrust 处理 → DOM 更新指令
    │
    ▼
jrust-runtime 执行更新 → jrust-browser 渲染 → 屏幕呈现
```
